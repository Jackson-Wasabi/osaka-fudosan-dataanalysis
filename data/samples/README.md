# data/samples — 公開用サンプルデータ

リポジトリ閲覧者がデータ構造を理解できるようにするための**少量の抜粋**です。
分析に使う元データ（raw）は再配布規約への配慮からリポジトリに含めていません。
同一データは `scripts/download_transactions.py` で再取得できます。

## osaka_condo_seiyaku_sample.csv

- **出典**: 不動産情報ライブラリ（国土交通省）「不動産価格（成約価格）情報」
  https://www.reinfolib.mlit.go.jp/
- **抜粋元**: 大阪府・中古マンション等・成約価格情報・2025年第1〜第4四半期（全11,094行）の先頭10行
- **取得日**: 2026-06-12
- **文字コード**: UTF-8（元データは cp932。閲覧しやすさのため変換済み）

## GIS データ（駅・公示地価）について

- 駅情報（国土数値情報 N02）・公示地価（同 L01）の加工済みサンプルは、
  Step 3（投入前確認）で `data/gis/processed/` を作成した後に追加予定。
- 国土数値情報は出典明示により利用可能:
  「国土数値情報（鉄道データ・地価公示データ）」（国土交通省） https://nlftp.mlit.go.jp/ksj/
