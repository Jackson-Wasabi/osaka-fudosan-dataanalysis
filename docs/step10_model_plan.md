# Step 10 モデル設計レビューと実行計画（予測㎡単価モデル）

作成: 2026-06-15 / 状態: 着手前のレビュー（未実装）。実装時に結果で更新する。

## 0. 目的
面積・築年・徒歩・駅相場などから「妥当な㎡単価」を予測し、実績との乖離で割安候補を一次スクリーニングする。
本モデルは**絶対的な適正価格の算定ではなく、駅相場に対する相対割安の検出器**として用いる。

## 1. モデル比較設計（PART 17）
- 目的変数: `log_price_per_sqm`（価格の歪みを抑えるためログ学習、評価時は円/㎡へ戻す）
- 学習/評価: `model_eligible AND split='train'` で学習、`split='test'`（未来）で評価＝ウォークフォワード
- リーク防止: 駅中央値㎡単価は各Fold訓練期間限定（D-009/D-012）

| モデル | 特徴量 | 狙い |
|---|---|---|
| Baseline | 駅中央値㎡単価のみ（モデルなし） | 比較基準 |
| Model A | 面積・築年数・徒歩分 | 基本特徴量の効果 |
| Model B | A + 改装済み + 新耐震 | 建物情報の効果 |
| Model C | B + 駅中央値㎡単価 | 駅相場の効果（採用候補） |
| Model D | C + 公示価格 | 改善しないことを実証→v9方針（予測に入れない）の根拠 |
| Model E | 木モデル(BOOSTED_TREE)・Cと同特徴量 | 線形の取りこぼし量を測る比較用 |

- Fold戦略: Fold A（本番・訓練2021-2024/テスト2025）で全モデル比較。採用 Model C を Fold B（訓練2021-2023/テスト2024）でも作り安定性確認。

## 2. 実務レビューで挙がった懸念点と対策（7項目）
面接・社内説明で「弱点を自分で言える」ようにするための記録。スクリーニング用途では絶対精度より**相対ランキングの頑健性**を主張する立て付け。

1. **市場の値上がりがモデルに入っていない（最重要）**
   - 懸念: 時間軸の特徴量が無く、過去で学習→2025予測すると上昇分を取りこぼし、2025予測が系統的に低く出て全体が「高い(+乖離)」に偏る。駅中央値も訓練期間（過去）由来で同様。
   - 対策: 一様バイアスなら順位は保たれる。まずテスト期間の符号付き誤差でバイアスを定量化→ (a) 割安閾値を残差分布から決める / (b) 後段キャリブレーション / (c) 時点インデックス項追加 から選ぶ。
2. **ログ逆変換の下振れ（Jensen）**
   - 懸念: `EXP(平均log) < 平均price` で予測が下振れ、MAE/MAPEと乖離基準がずれる。
   - 対策: Duanのsmearing補正（`AVG(EXP(残差))` を乗じる）。補正前後を併記。
3. **面積の5㎡丸め × 小型住戸（20-60㎡対象）**
   - 懸念: 20㎡で±2.5㎡=±12%の面積誤差→㎡単価が同率でブレる。誤差の床がデータ粒度で決まる（モデルの責任でない）。
   - 対策: 面積帯別にMAE/MAPEを出し「小型の誤差はデータ粒度由来」と切り分け。極端値は outlier_pps_flag で管理済み。
4. **station_median は目的変数の集約＝ほぼ循環特徴量**
   - 懸念: Model C は「駅相場＋微調整」化し乖離≒駅内乖離。訓練行は自価格が自駅中央値に入る軽リーク。小規模駅は中央値が不安定。
   - 対策: 評価はテスト期間ベースで指標は膨らまないため優先度低。用途を「相対割安の検出」と明言。必要なら訓練の駅中央値を leave-one-out 化。少件数駅は信頼性スコアで減点済み。
5. **評価指標がMAPE偏重**
   - 懸念: %誤差は低㎡単価で発散・過小予測に非対称。
   - 対策: MAE・中央絶対％誤差・±10%/±20%以内の的中率・RMSE を併用（MAPEは参考）。
6. **線形回帰のみ＝非線形・交互作用の取りこぼし**
   - 懸念: 築年減価・徒歩効果・立地交互作用は非線形。残差に系統パターンが残る。
   - 対策: 説明用に線形を主役にしつつ Model E（木モデル）で取りこぼし量を把握。残差を築年帯・徒歩帯・区で可視化。
7. **2025テストデータの報告ラグ**
   - 懸念: 後半四半期が過少だとテスト集合が偏る。
   - 対策: 着手前に2025の四半期別件数を確認し、偏れば「テストは2025前半まで」等を検討。

## 4. フェーズ2後の判断ポイント（対策1の設計判断）
測定した値上がりバイアスを見て選ぶ:
- (a) そのまま＋割安閾値を残差分布から決める（順位は頑健・基本）
- (b) 予測を後段キャリブレーション（モデル仕様は不変）
- (c) 時点インデックス項を追加（v9特徴量から踏み込む・乖離の解釈が変わる）

## 4.5 フェーズ0 診断結果（2026-06-15 実施・sql/bqml/00_pre_checks.sql）

母集団: 年次は fold='A'、特徴量/分布/相関は train(2021-2024)・model_eligible。

- **⑦ 報告ラグ: 問題なし** → テストは2025通年。2025四半期件数 586/547/494/479（過去年より多い）。
- **① 値上がり: 確認・無視できない（最重要）**。生中央値 390k→511k/㎡（2021→2025・+31%）。面積帯固定(20-30) 465k→600k(+29%)、主要13駅固定 450k→567k(+26%)＝**構成変化でなく実上昇**。訓練の駅中央値480k vs 2025実績511k=**+6.5%ギャップ**。→ モデルは2025を約6-7%低く予測する系統バイアス。**時点補正が必要**。
- **A ログ変換: 妥当**。歪度 pps=1.76 → log=-0.18（ほぼ対称）。log採用を裏付け。
- **B 二変量**: 相関 築年-0.81(支配的)/駅中央値+0.66/徒歩-0.33/面積-0.05。築年帯中央値 680→600→466→316→300→216k で**非線形**。→ 符号想定通り＋木モデル比較の価値を裏付け。
- **C 共線性【重要】**: corr(駅中央値, 公示価格)=**0.40のみ**（強い重複ではない）。→ v9の「公示価格は駅相場と重複」前提は**事前には弱い**。**Model D は決め打ちせず実測**し、改善しても「二重カウント回避でスコア側に回す設計判断」として説明する立て付けに修正。他特徴量間は低相関で問題なし。
- **D 分布シフト**: 軽度。2025は小型(20-30㎡)が28%（訓練22%）。築年・徒歩の中央値は同じ。許容。
- **F 学習衛生**: 外れ値pps=0件（scopeで除外済）。駅別訓練件数は175駅中95駅(54%)が20件未満＝駅中央値が多くの駅で不安定→信頼性スコアの減点が重要。

### フェーズ1着手前の確定事項
1. テスト期間=2025通年　2. log_price_per_sqm 採用確定　3. 時点バイアス約6-7%は要補正（フェーズ2で残差実測→キャリブレーション、閾値は残差分布から）　4. 木モデル(Model E)を作る　5. **Model Dは結論を決め打ちせず実測**（land相関0.40でnarrativeを「重複」→「二重カウント回避の設計判断」に修正）　6. 外れ値除外・log採用は追加対応不要

## 4.6 フェーズ2 評価結果（2026-06-16 実施・sql/bqml/02_evaluate.sql）

Fold A・テスト2025（n=2,097）・model_eligible。log逆変換は Duan smearing 補正。主指標 hit10（±10%以内的中率）。

| モデル | hit10 | hit20 | MdAPE | MAE円/㎡ | 符号付バイアス平均 |
|---|---|---|---|---|---|
| Baseline(駅中央値) | 0.176 | 0.356 | 0.286 | 186,117 | -0.040 |
| A(面積+築年+徒歩) | 0.267 | 0.486 | 0.209 | 135,409 | -0.067 |
| B(A+改装+新耐震) | 0.269 | 0.482 | 0.209 | 135,611 | -0.067 |
| **C(B+駅中央値)** | **0.269** | **0.540** | **0.183** | **120,567** | **-0.093** |
| D(C+公示価格) | 0.269 | 0.537 | 0.188 | 120,655 | -0.095 |
| E_tree(BOOSTED) | 0.269 | 0.535 | 0.186 | 123,853 | -0.089 |
| F_time(C+時点index) | 0.329 | 0.586 | 0.161 | 101,093 | +0.042 |

**判断（フェーズ2後の対策1の答え）**:
- 採用=**Model C**。D=公示価格は効かず（v9支持・スコア側へ）、E_tree=木で取りこぼし無く線形十分（いずれも実測で根拠化）。
- **Model F_time は集計指標で最良だが採用不可**。全市共通の単一時点傾きで2025を底上げした結果、空間構造バイアスを悪化（周縁・割安区の過大予測：平野 C+0.15→F+0.33／住吉 +0.20→+0.39／此花・住之江・東住吉 +0.26〜0.28、築古40年+ +0.09）。過大予測=「割安候補」の誤検出＝割安抽出ツールの一次欠陥。**一様バイアスは順位を保つが構造化バイアスは順位を歪める＝MAEが良い≠スクリーニングに良い**。
- 時点バイアス対処は (b)スカラー/(c)時点項とも空間構造が残る/悪化のため不採用。**(a)の精緻版＝割安乖離を「物件 vs その駅相場」で相対測定**し、一様な時点レベルと駅固定の空間レベルを乖離計算で相殺（2025候補同士の比較なら頑健・プロジェクト方針「駅相場に対する相対割安」と一致）。
- Fold A/B安定性（C）: A(2025) hit10 0.269/バイアス-0.093、B(2024) 0.318/-0.064。外挿が短いBが良く値上がり説と整合・大崩れ無し。
- 残差: 面積は小型(20-30㎡)最良・大型(50-60㎡)最差＝事前懸念「小型が誤差床」は逆。築古の構造残差はlimitationとして明記、旧耐震・築古はリスク減点側で吸収。
