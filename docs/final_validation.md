## 3. 分析ロジックの妥当性

- 割安度の向き: 相場より下振れ（`under_level` が負）ほど高スコア。✅向き正しい。
- 件数フィルタ: 駅取引 **10件以上のみ**ランキング対象（66駅）。少数データのまぐれ安値を抑制し、件数を信頼性スコアに加点。✅
- スコア配点（バランス型 主スコア）: 割安45 + 信頼25(件数15+IQR10) + 地価15 + 品質15 − リスク20、0〜100クリップ。`mart_opportunity_list.sql` の `score_balance` で実装確認。✅
- スコア内訳列が mart に存在（`sc_value` / `sc_reliability` / `sc_land` / `sc_quality` / `risk_penalty`）→ Tableau/PDFで内訳可視化が可能（A-1対応の素材あり）。
- 感度分析: 3配点（バランス45/25/15/15/-20・割安重視60/15/10/15/-20・リスク重視35/30/15/20/-30かつ旧耐震-30）。3中2回以上Top20で `stable_flag`（安定20駅）。✅
- 逆選択の自動除外: 平野は割安重視で1位（85.6）だが `high_risk_share=0.857`（旧耐震×改装不明）で本命★から除外。`rank_risk=33`。✅設計どおり機能。

## 4. 採用モデルと採用理由（説明できること）

- **採用 = Model C（B+駅中央値）**。MAE 120,567円/㎡、hit10 0.269、Baseline比約35%改善。
- Model F（C+時点index）は精度最良（MAE 101,093・hit10 0.329）だが**不採用**。理由: 単一時点の傾きで郊外を過大予測し「偽の割安」を作り、駅の順位を歪める（目的は予測精度でなく駅の順位付け）。区別バイアス図（model_bias_by_ward）で F−C が全20区ほぼ一律+14.5%シフト＝平均は直すが区ごとのバラつき＝順位の歪みは直さないことを確認済み。
- Fold安定性: Fold A（訓21-24/テスト2025）hit10 0.269 / Fold B（訓21-23/テスト2024）hit10 0.318。大崩れなし。

## 5. 注意事項（資料に必ず明記する）

- 成約価格データであり売出ではない。最終判断は現物・謄本・管理状態・権利関係の確認が必要。
- 対象は中古マンション・面積20〜60㎡・徒歩20分・築5〜60年に限定（市場の約32%）。全市場ではない。

- 取引件数が少ない駅は参考値（10件未満は除外）。
- 「割安＝相場より下振れ」は好機か衰退かを単独では判別できない。Step12の逆選択検証＋現物確認が前提。
