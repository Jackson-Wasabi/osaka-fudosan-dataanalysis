# 検証メモ（Step 12・上位駅の目視検証）

実施: 2026-06-16 / 対象: mart_opportunity_list 上位10駅 / 方法: **データ駆動の逆選択スクリーニング**（外部サイトの目視はせず、最終の現物・謄本確認は人に渡す前提）

## 0. 目的と立て付け

ランキング上位を額面で信じない（D-024）。上位駅が「本物の割安エリアか、データに写らない理由で安い逆選択か」を、手元データの**逆選択の指紋**（旧耐震率・大型偏り・築年・極端外れ値の集中・乖離分布の一様性）で定量的に裏取りし、人が次に動く先を仕分ける。

## 1. 上位10駅の3分類

| 分類 | 駅（順位） | 決め手 |
|---|---|---|
| ①本物っぽい割安エリア | 堺筋本町(1)・玉造(2)・大国町(4)・森ノ宮(7) | 築8〜17年と新しめ・旧耐震ほぼ0・極端外れ値なし・駅内乖離が一様に安い＝エリアごとソフト |
| ②要注意（割安シグナルが不確実）| **平野(3)** | 旧耐震86%・大型(45-60㎡)95%・築年中央52年・少数(21件)。割安シグナルが最も信用できないセグメント（築古・大型・周縁・薄商い）に集中→買い場でなく軟調/流動性低下/隠れ瑕疵の可能性 |
| ③要・現物確認 | 松屋町(5)・中之島(6)・谷町四丁目(8)・桜ノ宮(10)・難波(9) | 一部の極端外れ値が牽引／中之島は高額大型の個別出物／桜ノ宮は薄い(12件)／難波はそもそも割安でない(under_level +0.04) |

## 2. 駅別の所見

- **玉造(2)**: 最も有望。割安(-0.19)だが築8年と新しく旧耐震0・極端外れ値なし。新しめ在庫がエリアごと相場より安い＝本物のソフトエリア候補。※重複候補13%は留意。
- **堺筋本町(1)・大国町(4)**: 中心業務地で穏やかな割安(-0.12/-0.16)、分布がタイト（大国町は最割安物件でも-17%）＝一様な割安。クリーン。
- **平野(3)**: 割安シグナルを額面で信じない代表例。Model C は築年数を特徴量に含むため「古いから安い」とは断定できない（古さは既に減価済み）。重要なのは**割安が最も不確実なセグメント（築52年・大型95%・周縁・21件と薄商い）に集中**している点。古い周縁・少数取引では駅中央値の信頼性も低く、`割安 vs モデル`は買い場でなく軟調/流動性低下/隠れ瑕疵の疑いが濃い。よってメカニズムは断定せず、**リスク仕分けとして②要注意に降格**。バーゲン比率0.43も老朽大型に偏った見かけ。
- **松屋町(5)**: 駅水準はわずか割安(-0.07)。中央区55㎡が予測の半額(-70%)という1物件が牽引。事故物件/借地/再建築不可/データ誤りの疑い濃厚。要現物確認。
- **中之島(6)**: 高層タワー街。極端割安は新築大型の高額物件(52〜92万/㎡が予測比-38〜48%)。低層階/方位等の個別事情の可能性。要確認。

## 3. 横断的発見（重要）

**極端割安の個票（駅相対-30〜-70%）はほぼ全て outlier_pps_flag=0＝既存の外れ値フラグに無印。**
既存フラグは市全体のIQR（絶対）判定のため、**駅に対して相対的に異常な物件**をすり抜ける。松屋町の-70%物件が無印で残存。
→ (a) だから現物確認が必須。(b) 将来改善: 「駅相対の外れ値フラグ」を加えれば、この種の逆選択/データ誤りを自動で隔離できる（次バージョンの提案）。

## 4. ビジネスインパクト（確定値）

- 母数: 大阪市の中古マンション **2,097件 / 151駅**（2025年・scope内）
- スクリーニング: 調査対象を **上位10駅（物件309件＝全駅の約7%）** に圧縮
- 候補プール: 駅相対で15%以上安い **472件（22.5%）** を抽出
- 仕分け結果: 上位10駅のうち **約4駅が本物の割安エリア候補・1駅は逆選択・残りは要現物確認** ＝ スクリーナーが正しく機能

## 5. 人が次にやること（モデルの外）

1. ③の駅・物件（特に松屋町-70%・中之島の高額出物）の**謄本・現地・管理状態**を確認。
2. ①の駅（玉造・堺筋本町・大国町）を**重点ソーシングエリア**として売出リストを当たる。
3. ②平野は**老朽大型エリア**として扱い、個別の再建築可否・耐震を精査した上でのみ検討。

## 6. 限界

- 成約データのため個別物件は「買えない」（手法デモ）。本命は駅・エリアの優先度（D-024）。
- 階数・方位・正確な住所・リフォーム状態・心理的瑕疵は公開データに無い。本メモの「逆選択疑い」は確定でなく**人の確認を促すフラグ**。
