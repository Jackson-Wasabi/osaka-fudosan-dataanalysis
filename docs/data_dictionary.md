# データ辞書 (data_dictionary)

BigQuery `osaka-fudosan-dataanalysis.osaka_real_estate` の raw テーブルのカラム日本語対応表。
作成日: 2026-06-12 / 出典: 不動産情報ライブラリ・国土数値情報の公式定義に基づく。
※ BigQuery 上でも各列の説明欄（description）に同内容を設定済み（scripts/add_column_descriptions.py）。

## 1. raw_transactions（成約価格情報 47,386行）

元CSV（不動産情報ライブラリ）の日本語ヘッダーを、BigQueryの列名制約（「：」「（）」使用不可）のため英語名に置換している。列順は元CSVと完全一致（列ズレなしは sql/checks/01_raw_load_validation.sql で検証済み）。

| #  | BigQuery列名                 | 日本語カラム名     | 説明・値の例                  |
|----|------------------------------|--------------------|-------------------------------|
| 1  | kind                         | 種類               | 「中古マンション等」のみ      |
| 2  | price_category               | 価格情報区分       | 「成約価格情報」のみ          |
| 3  | city_code                    | 市区町村コード     | 5桁。例: 27102                |
| 4  | prefecture_name              | 都道府県名         | 「大阪府」のみ                |
| 5  | city_name                    | 市区町村名         | 例: 大阪市都島区              |
| 6  | district_name                | 地区名             | 例: 網島町                    |
| 7  | nearest_station_name         | 最寄駅：名称       | 例: 大阪城北詰 (注1)          |
| 8  | nearest_station_distance_min | 最寄駅：距離（分） | 例: 5 (注2)                   |
| 9  | trade_price_total            | 取引価格（総額）   | 円。例: 13000000              |
| 10 | layout                       | 間取り             | 全角。例: ２ＬＤＫ            |
| 11 | area_sqm                     | 面積（m2）         | 例: 55。5m2単位の丸め         |
| 12 | built_year                   | 建築年             | 例: 1980年。「戦前」あり      |
| 13 | building_structure           | 建物の構造         | 例: ＳＲＣ、ＲＣ、鉄骨造      |
| 14 | use_type                     | 用途               | 欠損多                        |
| 15 | future_use_purpose           | 今後の利用目的     | 例: 住宅。欠損多              |
| 16 | city_planning                | 都市計画           | 例: 近隣商業、商業            |
| 17 | building_coverage_ratio      | 建ぺい率（％）     | 例: 80                        |
| 18 | floor_area_ratio             | 容積率（％）       | 例: 400                       |
| 19 | trade_period                 | 取引時期           | 例: 2021年第1四半期           |
| 20 | renovation                   | 改装               | 「改装済み」or 空欄のみ (注3) |
| 21 | trade_circumstances          | 取引の事情等       | ほぼ欠損                      |

- **注1**: 表記ゆれあり（staging で正規化）。空欄率 0.3〜2.9%
- **注2**: 範囲表記が混在（「30分～60分」44件、「1H～1H30」1件、「1H30～2H」2件）。空欄率 6〜9%
- **注3**: 「改装済み」2,512件 / 空欄 44,874件。「未改装」という値は存在しないため、空欄は未改装か不明か区別できない（renovation_unknown_flag の設計に反映）
- 価格レンジ: 100万〜8.2億円 / 面積レンジ: 15〜1,960㎡（外れ値の分類は Step 5 EDA で実施）

## 2. raw_station_master_2025（国土数値情報 N02 鉄道・駅）

全国の駅（2025: 10,234行）。列名は国土数値情報の属性コードを小文字化したもの。

| BigQuery列名 | 日本語名         | 説明                                      |
|--------------|------------------|-------------------------------------------|
| n02_001      | 鉄道区分コード   | 路線の種類による区別（下記コード表）      |
| n02_002      | 事業者種別コード | 事業者による区別（下記コード表）          |
| n02_003      | 路線名           | 例: 御堂筋線、東海道線                    |
| n02_004      | 運営会社         | 例: 大阪市高速電気軌道                    |
| n02_005      | 駅名             | 例: 梅田、大阪                            |
| n02_005c     | 駅コード         | 緯度降順で付加した一意の番号              |
| n02_005g     | グループコード   | 乗換駅等の同一駅を束ねる (駅名寄せに有用) |
| lon / lat    | 経度 / 緯度      | 駅ジオメトリの代表点（簡易中点で算出）    |

- **鉄道区分コード（n02_001）**: 11=普通鉄道JR / 12=普通鉄道 / 13=鋼索鉄道 / 14〜16=モノレール・案内軌条式等 / 21〜=軌道（路面電車等）
- **事業者種別コード（n02_002）**: 1=新幹線 / 2=JR在来線 / 3=公営鉄道 / 4=民営鉄道 / 5=第三セクター

## 3. raw_land_price_2025（国土数値情報 L01 地価公示・大阪府）

標準地1地点=1行（2025: 1,687行×148列）。
全列の正式定義: [製品仕様書 KS-PS-L01-v3_3.pdf](https://nlftp.mlit.go.jp/ksj/gml/product_spec/KS-PS-L01-v3_3.pdf)

### 3-1. 分析の主要列

| BigQuery列名  | 日本語名              | 説明・例                                 |
|---------------|-----------------------|------------------------------------------|
| l01_001       | 行政区域コード        | 5桁。例: 27102                           |
| l01_002       | 用途区分（当年）      | 000=住宅地、005=商業地 等                |
| l01_003       | 連番（当年）          | 市区町村・用途区分単位                   |
| l01_007       | 調査年                | 例: 2025                                 |
| **l01_008**   | **公示価格（円/m2）** | 例: 208000 = nearest_land_price の源泉   |
| **l01_009**   | **対前年変動率（%）** | 例: 3.5 = land_price_change_rate に利用  |
| l01_010       | 選定状況              | 継続・新規等の地点異動情報               |
| l01_024       | 標準地名              | 例: 大阪都島                             |
| l01_025       | 所在並びに地番        | 例: 大阪市都島区毛馬町５丁目２３番       |
| l01_026       | 住居表示              | 例: 毛馬町５－６－２４                   |
| l01_027       | 地積（m2）            | 例: 185                                  |
| l01_028       | 利用の現況・大分類    | 住宅/店舗等 = 住宅系絞り込みに使用       |
| l01_047       | 周辺の土地利用の現況  | 例: 住宅、マンション等が混在する住宅地域 |
| l01_048       | 最寄り駅名            | バス停名等の場合あり                     |
| l01_050       | 最寄り駅まで道路距離  | 単位m。例: 750                           |
| l01_051       | 都市計画の用途区分    | 例: 2中専、商業                          |
| l01_053       | 都市計画区分          | 例: 市街化                               |
| l01_057 / 058 | 建蔽率 / 容積率       | 上限%。記載なし=0                        |
| lon / lat     | 経度 / 緯度           | 標準地の位置（Pointから算出）            |

### 3-2. その他の列グループ

| 列範囲      | 内容                                                               |
|-------------|--------------------------------------------------------------------|
| l01_004-006 | 前年の標準地番号（行政区域コード/用途区分/連番）。同一地点判定用   |
| l01_011-023 | 各属性の「前年からの変更有無」フラグ群                             |
| l01_030-046 | 形状・接道・施設等の詳細（間口比率、階層、前面道路、ガス・水道等） |
| l01_059-061 | 割増容積率、選定年次ビット等                                       |
| l01_062-070 | その他詳細属性（製品仕様書参照）                                   |
| l01_071以降 | 年次別公示価格系列（円/m2）。新しい列ほど近年。非選定年は空/0      |
